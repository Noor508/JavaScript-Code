Callback Hell

  1. Nested Callbacks stacked below one another forming a pyramid structure . (pyramid of Doom).
  2. This style of programmimg becomes difficult to understand & manage .
  3. if we want to run our instructions with some individual delay then we will use it.

Syntax

        asyncFunction1(() => {
            asyncFunction2(() => {
                asyncFunction3(() => {
                    // and so on...
                });
            });
        });
